TradingAgents-CN / 初始化说明

多智能体股票研究平台，本地启动路线与能力总览

本页基于当前仓库源码与文档整理，适合作为第一次接手项目时的入口。项目定位是学习与研究用途的 AI 股票分析系统，不提供实盘交易指令。

本机初始化状态

Python 默认版本偏低 当前 python3 是 3.9.6，项目要求 3.10+。本机已有 /opt/homebrew/bin/python3.10、Python 3.11 和 3.12 可用。
Node/npm 可用 当前 Node v25.5.0、npm 11.8.0，可启动 Vue 前端；项目要求 Node 18+。
Docker 未安装或不可用 docker 命令不存在。若要走推荐 Docker 路线，需要先安装 Docker Desktop 或配置 Docker CLI。

已从 .env.example 复制生成本地 .env。它只是模板副本，不包含真实密钥；启动分析前需要填入至少一个 LLM API Key，A 股数据建议再填 TUSHARE_TOKEN。

项目能力

多智能体股票分析

包含市场技术、基本面、新闻、社交媒体、研究员、交易员、风控与管理者等角色，形成从信息收集到投资观点生成的分析链路。

多市场覆盖

支持 A 股、港股、美股相关分析入口，前端也包含多市场股票查询、个股详情和历史分析页面。

多数据源

集成 Tushare、AKShare、BaoStock、Yahoo Finance、Finnhub，并提供数据同步、单股同步、多周期历史数据和数据源优先级配置。

多 LLM 提供商

模型目录覆盖 OpenAI、Anthropic、Google Gemini、DeepSeek、通义千问、OpenRouter、AiHubMix、Ollama、GLM、自定义 OpenAI 兼容端点。

现代化 Web 控制台

Vue 3 + Element Plus 前端包含仪表盘、股票分析、批量分析、任务队列、报告中心、自选股、筛选器、模拟交易、系统设置等页面。

后端 API 与实时通道

FastAPI 提供 REST API、SSE 和 WebSocket，用于任务进度、通知、队列、报告、配置、缓存、日志和数据库管理。

报告与导出

分析报告支持列表、详情、模块化内容读取，以及 Markdown、Word、PDF 等导出相关能力。

配置与用量治理

支持 LLM 厂家、模型目录、默认模型、数据源、数据库连接、缓存、Token 用量和成本统计的可视化管理。

模拟交易与学习中心

提供虚拟账户、下单、持仓、订单、重置等接口，以及 AI 基础、提示词、多智能体原理、风险局限和论文资料等学习内容。

架构与模块

目录 作用
app/ FastAPI 后端，包含认证、分析任务、报告、股票数据、配置、数据库、缓存、日志、调度、通知、模拟交易等路由与服务。
frontend/ Vue 3 + Vite 前端，默认开发端口 3000，开发环境代理 /api 到后端 8000。
tradingagents/ 核心多智能体图、分析师、数据流、工具、LLM 客户端、默认配置和股票研究流程。

cli/ 命令行交互入口，适合不启动前端时做快速分析或初始化。
scripts/ 数据同步、诊断、迁移、部署、启动、测试、数据库初始化等脚本集合。
docs/ 架构、部署、配置、数据源、LLM、故障排除、用户指南、发布说明和学习资料。

怎么启动

推荐：Docker

适合完整体验新架构，包含后端、前端、MongoDB、Redis。当前机器缺少 Docker，需要先安装 Docker Desktop。

cp .env.example .env
# 编辑 .env，填入 API Key
docker compose up -d

# 访问
# 前端：http://localhost:3000
# 后端：http://localhost:8000
# API 文档：http://localhost:8000/docs
开发：本地前后端

适合改代码。需要 MongoDB 和 Redis 已在本机或远程可用，并在 .env 中配置连接。

/opt/homebrew/bin/python3.10 -m venv env
source env/bin/activate
python -m pip install --upgrade pip
pip install -r requirements.txt
pip install -e .

# 启动 FastAPI 后端
python -m app

# 另开终端启动前端
cd frontend
npm install
npm run dev

访问 http://localhost:3000，前端会把 /api 代理到 http://localhost:8000。

备用：Streamlit

适合快速试旧版界面，默认端口 8501。注意根目录没有 start_web.py，更稳妥的是直接运行 Streamlit 或 web/run_web.py。

source env/bin/activate
python -m streamlit run web/app.py

# 或
python web/run_web.py
CLI

适合命令行测试多智能体流程。

source env/bin/activate
python -m cli.main

必须配置

配置项 说明 建议
DEEPSEEK_API_KEY / DASHSCOPE_API_KEY / OPENAI_API_KEY / GOOGLE_API_KEY 至少配置一个大模型密钥，否则无法完成智能体分析。 国内网络优先试 DeepSeek 或通义千问；已有 OpenAI 兼容网关也可走自定义端点。
TUSHARE_TOKEN A 股数据源推荐配置项，部分基础数据、财务数据和同步能力依赖它。 没有 Token 时可先依赖 AKShare 做轻量测试，但数据完整性会受限。
MONGODB_* / REDIS_* 新架构用于报告、任务、配置、缓存、通知、用量统计等持久化能力。 Docker 路线自动起服务；本地路线需要自己启动或改成远程连接。
JWT_SECRET 后端认证使用的签名密钥。 生产或长期使用不要保留默认值。

首次使用建议

  1. 先填 .env 中的 LLM Key，并确认后端能访问对应模型服务。
  2. 如果走新架构，先启动 MongoDB 与 Redis，再启动 python -m app 和 npm run dev。
  3. 打开前端后先进入系统设置，检查模型、数据源、数据库和缓存配置。
  4. 先用研究深度 1、少量分析师测试，例如 A 股 000001、美股 AAPL、港股 0700.HK。
  5. 做正式分析前先完成股票基础数据或单股数据同步，否则报告可能缺失行情、财务或新闻材料。
